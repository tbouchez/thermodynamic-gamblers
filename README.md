# Thermodynamic gamblers: why warming favours microbes that accelerate it

Code and data for:

> Dev S., Desmond-Le Quéméner E., Katipoğlu-Yazan T., Bouchez T. "Thermodynamic gamblers: why warming favours microbes that accelerate it." (submitted 2026).

## Overview

This repository contains the data and analysis code underlying a thermodynamic model of microbial growth applied to 1,054 strains across Bacteria, Archaea, and Eukarya. The model decomposes the temperature response of growth rate µ(T) into a kinetic term (Eyring: thermal degradation rate) and a thermodynamic term (Crooks: net assembly probability), using three fitted parameters: E_r (activation enthalpy of thermal degradation), E_h (thermodynamic regime), and T_max (upper thermal limit). The analysis reveals that heterotrophic microorganisms accelerate more than phototrophic ones under warming, constituting a positive feedback on climate change.

## Data files

| File | Description |
|------|-------------|
| `07a` | Growth rate measurements (µ vs. T) for 1,054 strains with taxonomic and trophic annotations |
| `07b` | Three-parameter model fits (E_r, E_h, T_max) obtained by differential evolution (DE). **This file is the authoritative parameter source — see note below** |
| `07c` | Independent Bayesian MCMC validation (10 chains × 50,000 iterations per strain) |
| `07d` | Comparison of the thermodynamic model against classical thermal models (Ratkowsky, CTMI, etc.) |

### Important note on `07b`

The file `07b_DE_results_1054_strains_v31.xlsx` contains the definitive fitted parameters. It was generated by `09_fit_growth_model_DE_v31.py` and subsequently verified against MCMC posteriors. A dormant formula bug was later identified in the fitting script that does not affect the existing output file but would produce incorrect results if the script were re-run. **Do not regenerate `07b`; use the provided file as-is.** The MCMC validation (`07c`) independently confirms the DE point estimates.

## Reproduction instructions

### Prerequisites

- Python ≥ 3.8
- Install dependencies: `pip install -r requirements.txt`

### Running the scripts

Scripts are designed to be run from the repository root directory.

**1. Model fitting (optional — output already provided)**

```bash
# Can be skipped: 07b is provided and must not be regenerated (see note above)
python scripts/09_fit_growth_model_DE_v31.py
```

**2. Main figures (Fig. 1–4)**

```bash
python scripts/08a_main_figures_generation_v31.py
```

**3. Extended Data figures (ED Fig. 1–10)**

```bash
python scripts/08b_ED_figures_generation_v31.py
```

**4. Bootstrap robustness analysis for α**

```bash
python scripts/10_bootstrap_robustness_alpha_v31.py
```

This performs 10,000 bootstrap replicates to assess the robustness of the per-capita acceleration ratio α (heterotrophs vs. phototrophs) under +4°C warming from a 15°C baseline.

## Model equation

```
µ(T) = (kB·T/h) · exp(−Er/RT) · [exp(Eh/R · (1/T − 1/Tmax)) − 1]
         \_________________/     \____________________________________/
          Eyring: thermal              Net assembly probability
          degradation rate k−                (Crooks)
```

where kB is Boltzmann's constant, h is Planck's constant, R is the gas constant, and T is in Kelvin.
